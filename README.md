# flexbox-mixin

a LESS mixin for css FlexBox

* IE 10 +
* iOS 3.2 +
* Safari 3.1 +
* Android 2.1 +
* Chrome 21 +
* Firefox 2 +
* Opera Mobile 12.1 +

Compatibility:

* IOS Safari 3.1 +
* Android 2.1 +
* Chrome 21 +
* Firefox 2 +
* Opera Mobile 12.1 +
* UC Browser for Android 9.9+
